# Tech Events Dashboard 🚀

A streamlined, modern web interface designed for managing upcoming technology events and user registrations.

## 📁 Included Files

*   **`event1.html`** – A clean, responsive dashboard layout displaying upcoming tech events and schedule details.
*   **`register1.html`** – A user-friendly registration form for attendees to sign up for events.

## 🛠️ Tech Stack

*   **HTML5** – Structured semantic layout.
*   **CSS3** – Modern styling and responsive design.

## 🚀 How to Run Locally

1. Open your terminal or command prompt.
2. Clone this repository:
   ```bash
   git clone [https://github.com/lazykishore/tech-events-dashboard.git](https://github.com/lazykishore/tech-events-dashboard.git)
   Open either event1.html or register1.html directly in any modern web browser (Chrome, Edge, Firefox) to view the pages!
